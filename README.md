# Superstore Predictive Analytics Dashboard

## Project Overview
This project analyzes historical Superstore sales data and predicts future sales trends using Power BI and Python.

## Tools Used
- Power BI
- Python
- Pandas
- ARIMA Forecasting

## Dataset
Superstore Sales Dataset

## Dashboard Features
- Sales Analysis
- Profit Analysis
- Customer Analysis
- Product Analysis
- Sales Forecasting

## Key Insights
- Best Selling Products
- Most Profitable Products
- Loss-Making Products
- Regional Performance
- Future Sales Forecast
